# 세니얼 Lite

세니얼 Lite는 쇼츠용 영상에 자막을 넣고, 장면별 이미지 프롬프트를 정리해서 WEBM 영상으로 내려받는 무료 배포 버전입니다.

## 바로가기
- 센텀하이 카페: https://cafe.naver.com/btbt7

## 사용 방법
1. `index.html`을 더블클릭해서 실행합니다.
2. 주제 또는 대본을 입력합니다.
3. 음성 파일을 넣고 음성 길이를 적용합니다.
4. 장면을 나누고 자막을 확인합니다.
5. 미리보기로 싱크를 확인합니다.
6. WEBM 다운로드를 누릅니다.

## API 키 안내
- API 키 기본값은 비어 있습니다.
- 사용자가 직접 GPT, Gemini, Claude 키를 입력해야 합니다.
- API 사용 요금은 각 서비스 정책을 따릅니다.
- 배포자는 사용자의 API 키를 수집하지 않습니다.

## 배포 안내
무료 배포용입니다. 수정 후 재배포하거나 제작자명을 제거해서 배포하지 마세요.

## 주의사항
- 브라우저 환경에 따라 WEBM 다운로드 방식이 다르게 동작할 수 있습니다.
- MP4 변환은 브라우저에서 불안정할 수 있어 WEBM을 우선 사용합니다.
- 자막 싱크는 음성 파일 길이를 기준으로 계산됩니다.
