# 세니얼 Senial - GitHub Pages 테스트용

## 업로드 파일
- index.html
- style.css
- app.js
- README.md

## GitHub Pages 사용법
1. 새 저장소 생성: `senial`
2. 위 4개 파일 업로드
3. Settings → Pages
4. Branch: `main` / root 선택
5. 생성된 주소로 테스트

## 포함 기능
- 올자동 모드
- 자막 직접 편집: 1줄/2줄/3줄 이상 자유 입력
- 이미지 배치: 자막 1칸/2칸/3칸/직접 설정
- 장면별 이미지 직접 업로드
- 장면별 AI 이미지 생성
- 첫 이미지 썸네일 생성기
- 기본 이미지 움직임: 확대 → 위로 → 아래로 반복
- 장면별 움직임 수정
- 자막 크기 조절
- 영상 빠르기 선택 및 직접 입력
- 브루형 중앙 미리보기

## 주의
브라우저에서 API 키를 직접 쓰는 구조는 테스트용입니다. 실제 서비스/판매용은 서버형으로 바꾸는 게 안전합니다.
